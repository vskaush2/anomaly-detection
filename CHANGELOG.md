# Changelog

## 0.1.0 (2026-07-18)

Initial release.

- **V_DIFF (v1)** — DBSCAN-based bad change detector with slope and GINI post-filters
- **V_DIFF_V2** — Gate-based detector with floor, percentile rank, spike suppression, correlated check, and robust z-score gates
- Type hints on the full public API
- 23 tests covering all detection gates
- Demo notebook with synthetic data scenarios
